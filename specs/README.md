# Specs

Specs para features grandes, milestones o spikes que necesiten plan antes de implementacion.

## Sugerencia Inicial

Crear una primera spec para el prototipo Tauri:

- tray;
- global shortcut;
- clipboard text capture;
- SQLite persistence;
- searchable picker;
- copy/paste selected item.

## Estructura Recomendada

```text
specs/<feature>/
├── spec.md
├── plan.md
├── research.md
├── data-model.md
├── quickstart.md
└── tasks.md
```
